# space-invaders

Классическая игра Space Invaders на C++17 с использованием ncurses.

## Сборка

### Требования
- C++17 компилятор (g++)
- CMake
- Ninja
- ncurses library

### Установка зависимостей (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install g++ cmake ninja-build libncurses5-dev libncursesw5-dev
```
